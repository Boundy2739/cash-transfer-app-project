# Cash Transfer Web Application

A web-based cash transfer system that allows users to send and receive money digitally.

## Overview

This project is a web application built using:

- **HTML & CSS** – Frontend design and layout  
- **PHP** – Backend logic and server-side processing  
- **MySQL** – Database management  

The system allows users to register, log in, transfer funds, and view transaction history.

---

## Features

- User registration and login system
- Secure authentication
- Account balance management
- Send and receive money
- Transaction history tracking
